## 💻 Projetos em destaque

### Zyntra
- **O que é:** Um sistema de gestão para clínicas e consultórios médicos.
- **O que eu implementei:** Construção do front-end e colaboração no desenvolvimento do backend, participando da integração entre interface e dados em tempo real.
- **Tecnologias:** Next.js na interface e Convex no backend. A escolha do Convex foi para garantir que o sistema atualizasse os dados (como status de pacientes na recepção) em tempo real sem precisar recarregar a página.
- **Resultado:** Uma plataforma funcional onde médicos e recepcionistas trabalham sincronizados instantaneamente.

### Nonno English
- **O que é:** Uma plataforma de gestão acadêmica e financeira para o ensino de idiomas.
- **O que eu implementei:** Desenvolvi um portal para o aluno (acesso a materiais e tarefas) e um painel administrativo para o professor (gestão de mensalidades, alunos e aulas).
- **Tecnologias:** React no front-end e Supabase (PostgreSQL) cuidando do banco de dados e da autenticação. Para o painel financeiro do professor, usei a biblioteca Recharts para montar os gráficos.
- **Resultado:** Um sistema centralizado que elimina a necessidade de usar planilhas separadas para gerenciar as finanças e os alunos.
